Colaborar com o programa

Você pode colaborar com o programa das seguintes maneiras:

* Comunicar erros de programa ou sugerir mudanças ou melhorias. Envie suas sugestões para o programa de e-mail (tangram.peces @ gmail.com).
* Enviando os dados que você criou. Basta enviar os arquivos de dados (com a extensão .fig) para o programa de correio. Após a revisão, os valores serão incorporadas em novas versões do programa.
* Traduzindo o programa. Se você conhece uma língua que não é traduzido, você pode fazer a tradução. Apenas é necessário editar alguns arquivos de tradução e salvá-lo para o idioma (locale_<codigo lingua>.lo). Então traduzir as frases que estão à direito do sinal de igual em cada linha do arquivo. Por fim, enviar o arquivo para o programa de correio.
