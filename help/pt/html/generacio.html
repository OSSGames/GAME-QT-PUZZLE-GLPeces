Geração de novas figuras

O programa pode gerar novas figuras. Esta opção só está disponível para algumas das modalidades do jogo. Podem fazer com que o programa gere uma figura com a opção de menu  Gera figuras -> Gera figura (ou bem com a combinação de teclas  Ctrl+R). O programa gerará uma figura nova e poderão tentar resolvê-la inmediatament.

Com a opção Gera figuras -> Gera figuras (ou bem com a combinação de teclas Ctrl+Q) poderão gerar figuras em massa (até 300). Só têm que introduzir o número de figuras que desejam gerar.

O processo de geração é bastante aleatório: mesmo assim, é possível que se gere alguma figura repetida. Pelisso, com a opção de menu Gera figuras -> Comprova figuras poderão revisar as figuras geradas. Com este ecrã podem fazer com que o programa comprove se há figuras repetidas entre as figuras geradas ou bem se alguma das figuras geradas já está na coleção das figuras do programa. Após fazer a comprovação, o programa marca automaticamente as figuras repetidas para eliminá-las. Podem marcar outras figuras (por exemplo alguma em que as peças não estejam colocadas formando uma única figura) com o botão de eliminação ().

A todas as figuras geradas se lhes atribui o nome «AUTO», seguido de um número gerado a esmo. Podem fazer com que o programa renumere correlativamente as figuras de forma automática com o botão .

Uma vez feitos as mudanças, podem guardar os dados em um arquivo. Para jogar com as figuras geradas, só têm que usar a opção de menu Aplicação->Carrega figuras de um arquivo e selecionar o arquivo desejado.
