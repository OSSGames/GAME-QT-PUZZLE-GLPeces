PRESENTACIÓ

Les característiques principals del programa Peces son:

* 40 modalitats de joc, set de les quals inventades per l'autor del programa.

* Quatre nivells de dificultat de joc.

* La possibilitat de crear noves figures i desar les dades a un arxiu

* Cerca de coincidències en les figures.

* Creació de concursos.

Consultau la secció Descripció del joc i la de menús per conèixer el funcionament del programa.
