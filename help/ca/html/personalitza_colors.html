Personalització del color de les peces

El color de les peces del joc és assignat de forma aleatòria pel programa. Podeu personalitzar el color de cada peça clicant (amb el botó esquerra del ratolí) a sobre a la vegada que manteniu pitjada la tecla de les majúscules (Shift). A continuació només cal que seleccioneu el color desitjat. El programa desa aquesta selecció automàticament.

Si voleu eliminar un color personalitzat o bé canviar el color assignat pel programa per un altre color seleccionat aleatòriament, només heu de clicar-hi a sobre amb el botó de la dreta del ratolí a la vegada que manteniu pitjada la tecla de les majúscules (Shift).
