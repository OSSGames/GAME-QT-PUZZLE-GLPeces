Col·labora amb el programa

Podeu col·laborar amb el programa de les següents formes:
  • Comunicant les errades del programa o bé fent suggeriments de millora. Adreçau els vostres suggeriments a l'adreça de correu del programa (tangram.peces@gmail.com).
  • Enviant les figures que heu creat vosaltres. Només cal enviar els arxius de figures (amb l'extensió .fig) a l'adreça del programa. Es revisaran les figures i s'incorporaran a les noves versions del programa.
  • Traduint el programa. Si feu servir una llengua a la qual no estigui traduït el programa, feu-ne la traducció. Només cal que editeu algun dels  arxius de traducció, ho desau amb el nom corresponent a la vostra llengua (locale_<codi ISO 639-1 de la llengua>.lo) i a continuació traduïu les frases que es troben a la dreta del signe =. En haver acabat, ho enviau a l'adreça de correu del programa.
