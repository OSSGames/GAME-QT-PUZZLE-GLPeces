Menu du programme

Vous avez ici la description des fonctionnalités des différentes options de menu du programme.

Programme

Choisir un fichier perso de figures. Avec cette option vous pouvez charger le fichier des figures créées par vous.

Charger les figures du programme. Cette option charge les figures distribuées par le programme.

Réinitialiser le jeu en cours. Cela remet à zéro les compteurs (l'horloge, le numéro des figures résolues et le score) correspondants à la modalité de jeu et au niveau actuels.

Réinitialiser tout. Cela remet à zéro les compteurs de toutes les modalités de jeu pour tous les niveaux.

Sortir. Cela ferme le programme.

Jeu

Modalité de jeu. Dans cette option vous avez la liste de toutes les modalités de tangram disponibles (33 jeux). Pour choisir une modalité, cliquez sur son nom.

Niveau de jeu Ici vous choisissez l'un des quatre niveaux de jeu disponibles (Enfant, Débutant, Intermédiaire ou Expert).

Création de nouvelles figures. Le programme permet la création de nouvelles figures. Pour cette fonctionnalité vous avez plusieurs options. Consultez la section de l'aide pour la Création de nouvelles figures.

Placer une pièce. Cette option du menu s'active quand on commence la résolution d'une figure. Si vous l'avez activée - en cliquant sur une pièce - le programme placera cette pièce dans sa position correcte. En utilisant cette option, le programme déduit des points pour chaque pièce placée par lui. Cette option n'est pas disponible dans le niveau Expert.

Voir la solution. Le programme montre la solution de la figure en cours. Si vous utilisez cette option, vous n'aurez plus la possibilité de résoudre la figure.

Rechercher une figure par son nom. Cette option est activée quand vous ne jouez pas. Elle permet de chercher dans l'arbre des figures une figure par son nom. La recherche est sensible à la casse (différence entre minuscule et majuscule).

Proposer une figure. Avec cette option, le programme choisit au hasard une figure parmi celles qui sont disponibles dans l'arbre des figures.

Concours

Cette option vous permet de créer des concours ou des activités en définissant un ensemble de figures. Consultez la section de l'aide Concours.
