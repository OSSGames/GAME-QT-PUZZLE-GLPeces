Collaborer avec le programme

 
Vous pouvez collaborer avec le programme de plusieurs façons:

* Communiquer les erreurs de programme ou suggérer des changements ou des améliorations. Envoyez vos suggestions par courriel (tangram.peces @ gmail.com).
* Envoyer les figures que vous avez créé. Il suffit d'envoyer les fichiers de figures (fichiers .fig) en les joignant à un courriel. Après examen, les figures seront intégrées dans les nouvelles versions du programme.
* Traduire le programme. Si vous connaissez une langue qui n'est pas traduite, vous pouvez faire la traduction. Il faut seulement modifier certains fichiers de traduction et les enregistrer pour la langue choisie (locale_<les deux lettres du code de la langue>.lo). Ensuite, traduire les phrases qui sont à droite du signe égal dans chaque ligne du fichier. Envoyez le fichier final en le joignant à un courriel.
