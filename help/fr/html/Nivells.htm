Le premier niveau (Enfant) est destiné aux plus petits, l'idée étant de leur faciliter les débuts dans le jeu. Pour résoudre une figure dans ce niveau, il suffit d'amener les pièces à l'intérieur de la silhouette : en les lâchant, le programme les place correctement.
