Génération de nouvelles figures

Le programme peut produire de nouvelles figures. Cette option est seulement disponible pour certains des modalités du jeu . Vous pouvez faire que le programme génère une figure avec l'option de menu Générer des figures -> Générer une figure (ou avec la combinaison des touches Ctrl+R). Le programme produira une nouvelle figure.

Avec l'option Générer des figures -> Générer des figures (ou avec la combinaison des touches Ctrl+Q) sera capable de ne produire des figures énormément (jusqu'à ce que 300). Vous avez à seulement introduire le nombre de figures que vous désirez générer.

Le processus de génération est assez aléatoire : cependant, il est possible qu'une figure répétée est générée. C'est pourquoi, avec l'option de menu Génère des figures -> Vérifie des figures vous pourrez réviser les figures générées. Avec cet écran vous pouvez faire que le programme vérifie s'il existe des figures répétées entre les figures générées ou bien si une des figures générées est déjà dans le recueil des figures du programme. Après avoir fait la vérification, le programme marque automatiquement les figures répétées pour les éliminer. Vous pouvez marquer d'autres figures (par exemple certaine dans laquelle les pièces ne sont pas placées en formant une figure unique) avec le bouton d'élimination ().

À toutes les figures générées on leur assigne le nom «AUTO», suivi d'un nombre généré au hasard. Vous pouvez faire que le programme renumérote corrélativement les figures de forme automatique avec le bouton.

Quand les changements ont été faits, vous pouvez garder les données dans un fichier. Pour jouer avec les figures générées, vous avez à seulement utiliser l'option de menu Programme -> Choisir le fichier perso des figures et à sélectionner le fichier désiré .
