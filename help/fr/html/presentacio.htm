Présentation

Les caractéristiques principales du programme Peces sont :

* 40 modalités de jeu dont 7 ont été inventées par l'auteur du programme.

* Quatre niveaux de difficulté de jeu.

* Possibilité de créer de nouvelles figures et de les enregistrer dans un fichier.

* Recherche de figures semblables.

* Création de concours.

Consultez la section Description du jeu et celle du Menu pour connaître le fonctionnement du programme.
