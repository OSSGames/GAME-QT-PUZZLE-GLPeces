Xeración de novas figuras

O programa pode xerar novas figuras. Esta opción só está dispoñible para algunhas das modalidades do xogo. Podedes facer que o programa xere unha figura coa opción de menú  Xera figuras -> Xera figura (ou ben coa combinación de teclas  Ctrl+R). O programa xerará unha figura nova e poderedes tentar resolvela inmediatament.

Coa opción Xera figuras -> Xera figuras (ou ben coa combinación de teclas Ctrl+Q) poderedes xerar figuras masivamente (até 300). Só tedes que introducir o número de figuras que desexades xerar.

O proceso de xeración é bastante aleatorio: aínda así, é posible que se xere algunha figura repetida. Por iso, coa opción de menú Xera figuras -> Comproba figuras poderedes revisar as figuras xeradas. Con esta pantalla podedes facer que o programa comprobe se hai figuras repetidas entre as figuras xeradas ou ben se algunha das figuras xeradas xa está na colección das figuras do programa. Despois de facer a comprobación, o programa marca automaticamente as figuras repetidas para eliminalas. Podedes marcar outras figuras (por exemplo algunha en que as pezas non estean colocadas formando unha única figura) co botón de eliminación ().

A todas as figuras xeradas asígnaselles o nome «AUTO», seguido dun número xerado ao azar. Podedes facer que o programa renumere correlativamente as figuras de forma automática co botón .

Unha vez feitos os cambios, podedes gardar os datos nun arquivo. Para xogar coas figuras xeradas, só tedes que usar a opción de menú Aplicación->Carga figuras dun arquivo e seleccionar o arquivo desexado.
