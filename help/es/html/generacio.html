Generación de nuevas figuras

El programa puede generar nuevas figuras. Esta opción sólo está disponible para algunas de las modalidades del juego. Podéis hacer que el programa genere una figura con la opción de menú  Genera figuras -> Genera figura (o bien con la combinación de teclas  Ctrl+R). El programa generará una figura nueva y podréis intentar resolverla inmediatament.

Con la opción Genera figuras -> Genera figuras (o bien con la combinación de teclas Ctrl+Q) podréis generar figuras masivamente (hasta 300). Sólo tenéis que introducir el número de figuras que deseáis generar.

El proceso de generación es bastante aleatorio: aún así, es posible que se genere alguna figura repetida. Por eso, con la opción de menú Genera figuras -> Comprueba figuras podréis revisar las figuras generadas. Con esta pantalla podéis hacer que el programa compruebe si hay figuras repetidas entre las figuras generadas o bien si alguna de las figuras generadas ya está en la colección de las figuras del programa. Después de hacer la comprobación, el programa marca automáticamente las figuras repetidas para eliminarlas. Podéis marcar otras figuras (por ejemplo alguna en que las piezas no estén colocadas formando una única figura) con el botón de eliminación ().

A todas las figuras generadas se les asigna el nombre «AUTO», seguido de un número generado al azar. Podéis hacer que el programa renumere correlativamente las figuras de forma automática con el botón .

Una vez hechos los cambios, podéis guardar los datos en un archivo. Para jugar con las figuras generadas, sólo tenéis que usar la opción de menú Aplicación->Carga figuras de un archivo y seleccionar el archivo deseado.
