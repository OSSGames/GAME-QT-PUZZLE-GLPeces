The program menus

He describes the program menu option functionality.

Aplicación

    Load figures from a file. With this menu option he can load the figures created by you from a file.

    Load program figures. This option loads the distributed figures with the program.

    Erase current game record. He puts to zero the counters (the clock, the solved figure number and the puntación) corresponding to the current game and level.

    Erase all records. He puts to zero the counters of all the forms of game and levels.

    Exit. Close the program.

Game

    Kinds of game. All available forms of tangram are in this option. Click on the corresponding name to select her.

    Game level. Select here the level of game between the four available levels.

    Create a new figure. The program allows the creation of new figures. Consult the section of the help Creation of new figures.

    Place piece. This menu option one activates when he begins one to play. If he selects it, when click on a piece, this will place one in his position in the figure. If he uses this option, the program will deduct part of the punctuation. This option is not available in the level Expert.

    View solution. He shows the current figure solution. If he uses this option, he will not yet be able to continue with the figure solving.

    Search the figure name. He allows to look for a figure by his name in the tree of figures. The searching distinguishes the lowercases from the uppercases.
 

    Suggest figure. This menu option selects one of the available figures randomly.

Competition

    This menu option allows the creation of competitions or activities selecting a figure group. Consult the help Competition section.
