Generation of new figures

The program can generate new figures. This option is only available for some of the modalities of game . You can do that the program generate a figure with the option of menu  Generates figures -> Generates figure (or with the combination of keys  Ctrl+R). The program will generate a new figure and will be able to try resolve it inmediatament.

With the option Generates figures -> Generates figures (or with the combination of keys Ctrl+Q) will be able to generate figures massively (until 300). Only you have to enter the number of figures that wish to generate.

The process of generation is quite random: still like this, it is possible that generate  some figure repeated. Therefore, with the option of menu Generates figures -> Checks figures will be able to review the figures generated. With this screen can do that the program check if there are figures repeated between the figures generated or if any of the figures generated already is in the collection of the figures of the program. After doing the checking, the program marks automatically the figures repeated to delete them. You can mark other figures (for example any in that the pieces are not placed forming an only figure) with the button of elimination ().

To all the figures generated assigns them  the name «AUTO», followed of a number generated at random. You can do that the program renumere correlatively the figures of automatic form with the button.

Once done the changes, can save the data in an archive. To play with the figures generated, only have to use the option of menu Application->Loads figures of an archive and select the archive wished.
