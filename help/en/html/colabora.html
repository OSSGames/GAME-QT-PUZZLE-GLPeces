Collaborate with the program

You can collaborate with the program of the following ways:

* Communicating program errors or suggesting changes or improvements. Send your suggestions to email the program (tangram.peces @ gmail.com).
* Sending the figures you have created. Simply send the files of figures (with the .fig) to the mail program. After review, the figures will be incorporated into new versions of the program.
* Translate the program. If you know a language which is not translated, you can do the translation. Only you must edit some of the translation files i saving it for the language (locale_<two letters language code>.lo). Then translate the sentences that are right of the equals sign in each line of the file. Finally, send the file to the mail program.
