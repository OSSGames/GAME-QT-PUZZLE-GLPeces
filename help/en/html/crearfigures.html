Creation of new figures

The program allows the creation of new figures. To create a new figure, not being in phase of solving of a figure is necessary. If he has already started the solving of a figure, he can stop the game clicking on the first item figure tree.

With the stopped game, he can build the new figure. Next, with the menu option Game-> Create new Shape-> Checks sample (or well with the combination of keys Ctrl+M) can check that figure sample and the outline are seen correctly. I could that some tightening is necessary. Give birth he erases the sample and the outline uses the new option of menu Game-> Create new Shape->Erase sample (or the combination of Crtl+E keys).

When he has the new figure, checking can if some figure similar already exists with the menu option Game-> Create new Shape->Search figure (or with the combination of Ctrl+F  keys.) The program will look for coincidences between the figures of the figure tree and the shaped by the pieces in the game zone. At the end, the program will report on the found coincidences.

Finally, you can saving the data of the new figure (or good with the combination of D Crtl keys) using the menu Game-> Create new Shape->Save the shape (or with the combination of Crtl+D keys). He will must assign a descriptive figure name and a difficulty level. You can select the saving file folder with the button .

He is able creating as many figures want. At any moment, using the menu option Program->Load figures from a file he can have the created figures in the tree of figures to play.

The figure data is saving in a text file. You can send the figure text files to the program author and they will be added to the program.
