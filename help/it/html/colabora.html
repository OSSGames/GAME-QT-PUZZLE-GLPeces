Collaborare con il programma

 
È possibile collaborare con il programma nei seguenti modi:

* Errori di programma per comunicare o proporre modifiche o miglioramenti. Inviare i vostri suggerimenti per il programma di posta elettronica (tangram.peces @ gmail.com).
* Invio i dati che hai creato. Basta inviare i file di dati (con l'estensione .fig) per il programma di posta electronica. Dopo il controllo, le cifre saranno integrate nelle nuove versioni del programma.
* Tradurre il programma. Se si conosce una lingua che non è tradotto, si può fare la traduzione. Solo è necessario modificare alcuni dei file di traduzione  (locale_<codigo di due lettere de la lingua>.lo). Poi tradurre le frasi che sono a destra del segno di uguale in ogni riga del file. Infine, inviare il file per il programma di posta elettronica.
